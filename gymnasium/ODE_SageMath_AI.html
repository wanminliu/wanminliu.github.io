is https://www.sagemath.org/.
One can install it or use it online Sage on CoCalc.
We use it to solve the ODE
(1.1)
. For simplicity, we use the cloud version
of SageMath on CoCalc.
Step 1. Visit
https://cocalc.com/features/sage

Step 2. After signing in, create a project
https://cocalc.com/projects

We call the project ODE-2024. Then we can create a SageMath Worksheet.
